Cài môi trường ảo và kích hoạt sau đó dùng lệnh python -m pip install -r requirements.txt để cài các thư viện trong đó (nếu không dùng môi trường ảo thì bỏ qua dòng này)
Các thư viện cần thiết bằng lệnh cài đặt: pip install fastapi uvicorn jinja2 python-multipart
Để chạy thì mở terminal và điền vào câu lệnh uvicorn main:app --reload sau đó mở trình duyệt và tìm đến http://localhost:8000/ để xem giao diện và sử dụng
